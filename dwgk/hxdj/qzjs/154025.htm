换届选举动员报告

发布日期:2011-10-28 02:28:09   来源:   

 

上海师范大学康城学校党支部
根据闵行区教育局党委的要求，本届党支部将实施“公推直选”形式进行换届改选工作。我们将严格按照教育局党委对换届改选方式实施“公推直选”工作提出的有关要求、规定和程序进行，在全校范围内进行。
为什么要推行“公推直选”？ “公推直选”的意义就在于：
1、公推直选把基层领导班子的“初始提名权、最终决定权”交给党员群众，坚持阳光操作，充分尊重民意，体现党员意志，尊重党员群众的主体地位，拓展党的基层干部选拔任用渠道，这是促进党内民主发展的一种新形式。
2、公推直选落实和保障了党员知情权；选举权；参与权和监督权。通过坚持“公开、公平、择优”的原则，进一步扩大党在学校的影响力，巩固党的群众基础，激发党员参与党内政治生活的热情，提高教职工对学校党支部领导班子的公认度，进一步增强学校党支部的创造力、凝聚力和战斗力。
3、公推直选能有效激发党员群众的民主意识，开拓基层党内民主发展的新境界，提升党的公信力，推动党内民主的发展。以党内民主的扩大，带动人民民主的扩大。真正落实十七大报告指出：要探索扩大党内基层民主多种实现形式。
为了落实好这项工作，9月14日教育局党委召开了直属党支部书记会议，就这项工作部署下达任务。我们学校于15日召开了支委会、支委扩大会，建立了领导小组，制定了实施方案。下面我代表换届改选领导小组把整个换届改选工作的程序向大家通报一下。
一、新一届支委人员设置和任职条件
（一）委员职数
新一届党支部设委员5人，其中书记1人，副书记1人。
（二）任职条件
党支部委员候选人必须是正式组织关系隶属于本支部的中国共产党正式党员，一般应具备大学专科及以上文化程度。
1、严格执行党的方针政策，善于学习思考，具有较强的政治理论水平。
2、热爱党的教育事业，教书育人，为人师表，
3、牢固树立党的观念、大局观念和群众观念，具有团队合作精神。廉洁自律，实事求是，公道正派。
4、热爱党务工作，有较强的组织管理能力和协调沟通能力。任副书记、书记，应当在下级岗位工作2年以上。
二、组织领导
（一）换届改选领导小组
组长：陈岳周 副组长：王若慧
组员：柯祥发 金燕连 沈叶萍、薛美香、黄宏锦、肖昌虎
三、换届选举方式
采用 “有候选人选举”方式，即群众、党员推荐，党员大会进行书记、副书记采用等额选举，委员采用差额选举的无记名投票直接选举。
四、实施步骤
换届改选工作分四个阶段
（一）宣传发动，制定方案阶段（9月14日—9月16日）
1、召开支委会和支委（扩大）会议，组织学习换届改选的“意见”和“实施方案”； 成立换届改选工作领导小组、工作小组。讨论工作安排与步骤，制订换届改选方案；
2、向局党委提出请示。
3、召开工作领导小组、工作小组联席会议，明确动员会和组织报名具体时间；
4、起草本届党组织工作报告。
（二）组织报名，民主推荐阶段（9月16日—9月23日）
1、召开动员会。（9月16日）
n    讲清公推直选的意义方法步骤，强调扩大民主，尊重党员的主体地位；讲清推荐的方法和程序，强调组织推荐的候选人可进入初步候选人名单；
n    制作下发党员名册推荐表，组织个人自荐和民主推荐；
2、资格审查。
n    工作小组回收汇总表格；
n    党组织对推荐人选逐一进行资格审查，重点是组织关系、任职条件等；
n    确定参加民主推荐的人选，印制候选人推荐表。
3、召开推荐大会。
n    工作小组发出候选人推荐表，组织党员群众推荐；
n    工作小组回收统计推荐表，上墙公布推荐票数情况；
n    领导小组将统计结果按序排列报局党委审查；
n    领导小组公布局党委审查确认的初步候选人+40%（7人）
（三）考察测评，公示人选阶段（9月23日---9月30日）
1、民主测评。参加对象：全体教职工。
2、个别访谈。访谈对象：中层以上干部代表，党员群众代表，（5—7人）
3、公示、介绍候选人预备人选的基本情况，20%（6人）公示时间为7天。
（四）正式选举，报请审批阶段（10月8日—10月15日）
1、会议准备
n    制订选举办法（草案）;
n    提出总监票人、监票人建议名单；
n    印制选举票；
n    确定选举大会应到会、有选举权的党员数；
n    会场布置：大会设置主席台，候选人上台就座；会场设立会标，突出主题；会场设有党旗；座位安排有序，党员和群众、正式和列席代表要分开，便于统计。
2、召开换届选举大会。邀请部分教职工代表列席。（10月12日）
会议由换届选举领导小组组长主持。
会议议程：
n    宣布开幕，全体起立唱《国歌》；
n    听取会议筹备工作和资格审查报告；
n    宣布局党委对我校正式候选人名单的批复；
n    宣读、通过大会选举办法（由工作人员宣读）；
n    通过总监票人、监票人员建议名单；
n    大会选举：
n    宣布选举结果；
n    奏《国际歌》，宣布会议闭幕。
3、选举结果报局党委，材料整理归档。
